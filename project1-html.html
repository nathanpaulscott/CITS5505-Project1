HTML - History and Evolution

What is HTML

HTML (Hyper Text Markup Language) is a simple text-based scripting language that is used to specify content and structure of client-side web sites. It works in harmony with CSS for sytle and Javascript for functionality. HTML essentially allows the specification of hierarchical tree structures where the nodes are known as tags.

The fundamental structure of all webpages are the 'HTML' tag as the root node, then 2 children:

  • head for meta data and setup information and code
  • body for the main content of the webpage


Tags are typically specified with an opening and closing pair of angle brackets with the displayed content found between, eg.

<div>
       some content here
</div>

Different tag types can have different structural properties such as <div> for containers, <h> and <p> for text, <img> for images and <button> for buttons. One of the key differentiators is whether a tag is typically displayed inline (on the same line) or block (on a new line). Most tags are block by default, but a few like <span> and <i> are inline by default. CSS is required to change this default behaviour.

Attributes of a tag like Id, name, style and event handlers are found in the opening tag, while the closing tag simply serve to close off that tag's block of code, eg.

<h5 id="test2" data-toggle="collapse" data-target="#p2" aria-expanded="false" aria-controls="p2">
       some more content
</h5>
HTML 1.0

HTML 1.0 was initially developed by physicist Time Berners-Lee in the 80's and realeased in 1989 as part of a technology suite aimed at enabling physicists to work together. Tim is also widely known as one of the creators of the internet. HTML 1.0 was used from 1989 to 1994, this was a very simple predecessor of modern HTML. It contained around 20 elements and it had no CSS. There were no tables or forms. It is interesting to note that HTML was heavily influenced by SGML which was a markup language used in-house at CERN, his place of work.

HTML 2.0

Created by an IETF working group to standardise HTML. It was released in 1995. HTML 2.0 became the standard for HTML against which all browsers were measured. Many of the improvements involved adjusting style, but also added tables and forms.

HTML 3.0

The World Wide Web Consortium (W3C) came to prominance in the mid 90's and overtook development of HTML as W3C had powerful commerical backers as opposed to the IETF. HTML 3.0 was the first HTML standard endorsed by W3C, it was released in 1997. It was supported by Netscape Navigator and Internet Exporer, two of the big players in browser technology at the time.

HTML 4.0

In 1999 HTML 4.0 was released which included the first iteration of CSS which marked the spearation of content and style. In 2000 HTML became an ISO standard as well.

HTML 5.0

Fianlly HTML 5.0 was released in 2014. This essentially brought the specification of HTML to completion, however minor releases have been made since then.
